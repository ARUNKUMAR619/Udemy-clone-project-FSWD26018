# Udemy Clone

A responsive clone of Udemy’s homepage built with *HTML, **CSS, and **FontAwesome* icons.

## Features

- Responsive navbar with logo, search bar, and icons  
- Hero section with overlay cards on images  
- Category pills for quick navigation  
- Recommended courses section with image cards  

## Tech Stack

- HTML5  
- CSS3 (Flexbox)  
- FontAwesome icons
